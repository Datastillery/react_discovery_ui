## Install Dependencies
`npm install`

## Run Unit Tests
`npm run test`

## Run Unit Tests in Watch Mode
`npm run test-watch`

## Lint the Code
`npm run lint`

## Start the UI Locally
`npm run start`

## Build Docker Image
`docker build . -t <image name>:<tag>`

## Running the Docker Image
`docker run -d --rm -p <port>:80 <image name>:<tag>`
